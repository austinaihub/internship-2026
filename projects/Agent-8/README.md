# Austin AI Hub – Internship 2026

This repository contains internship projects developed under the Austin AI Hub internship program.

## Purpose
- Applied AI experimentation
- Agentic systems and LLM-based workflows
- Research prototypes and proof-of-concept implementations
- Responsible and ethical AI practices

## Workflow
- Interns should **fork** this repository
- Create feature branches in their fork
- Submit Pull Requests (PRs) for review
- Do not push directly to `main` unless explicitly approved

## Structure (initial)
- `projects/` – Individual intern or team projects
- `experiments/` – Research experiments and prototypes
- `docs/` – Documentation, notes, and design write-ups

## Communication
All work should be documented clearly in PRs and issues.
